Google Setup (Required)

To use Saucebook, you need to create your own Google OAuth credentials. Follow these steps:

  1. Go to Google Cloud Console
  2. Create a new project
  3. Enable "Google Drive API" (APIs & Services → Library → Search → Enable)
  4. Create OAuth 2.0 credentials:
    • Credentials → Create Credentials → OAuth 2.0 Client ID
    • Select "Chrome Extension" or "Chromium Extension"
    • Enter Extension ID (from chrome://extensions)
  5. Copy your Client ID and paste below
